# nazrOS — Модули СТРАННО

Каждый модуль — отдельный Rust-крейт в `src/modules/`.
Загружается ядром через Plugin Registry как dylib.

## Очерёдность реализации

### Фаза 2 — РУКИ (3D-моделирование)
```
src/modules/руки/
├── Cargo.toml
├── src/
│   ├── lib.rs          # точка входа, регистрация в CORE
│   ├── viewport/       # Vulkan/WebGPU viewport
│   ├── mesh/           # полигональный меш
│   ├── sculpt/         # скульптинг + кисти
│   ├── materials/      # материалы + шейдеры
│   ├── library/        # библиотека форм
│   └── procedural/     # процедурный стек нодов
└── shaders/            # GLSL/WGSL шейдеры
```

**Ключевые возможности по скриншоту:**
- Режимы: скульптинг / полигональное / параметрическое / процедурное / инженерная точность
- Кисти: кисть, сгладить, вытянуть, надавить, сжать, гравировка, маска
- Слои скульпта (детали_лица, имплант, кабели, основа) + маски
- Библиотека форм: органические / механические / архитектура / биомеханика / символы
- Процедурный стек (шум поверхности → деформация формы → скульпт_деформация)
- Иерархия объектов справа (корень → группы → меши)
- Материалы: титан_мат, карбон_мат, керамика, нейро_гель, кожа_синт

### Фаза 5 — ШИНА (DAW)
```
src/modules/шина/          # C++ + Rust FFI
├── juce_engine/           # JUCE DSP движок (C++)
├── src/
│   ├── tracks/            # дорожки (драмы, бас, лиды...)
│   ├── synth/             # синтезатор (осцилляторы, фильтры, огибающие)
│   ├── mixer/             # микшер
│   ├── spatial/           # 3D-сфера пространственного аудио
│   ├── node_graph/        # узловая сеть звука
│   └── procedural/        # процедурная генерация (ритм, мелодия, аккорды)
```

**Кириллические форматы ШИНЫ:**
- `.звк` — звуковой проект
- `.синт` — пресет синтезатора
- `.ритм` — паттерн ритма (SEQ 16)
- `.прст` — аудио-пресет
