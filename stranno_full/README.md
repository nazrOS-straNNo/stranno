## Технологический стек

### Ядро (nazrOS CORE)
- **Язык**: Rust (основной), C++ (GPU-интенсивные модули)
- **Граф сцены**: собственная реализация на Rust
- **GPU**: Vulkan / Metal (macOS)
- **Форматы**: `.нзр` (сцены), `.звк` (звук) и другие кириллические форматы (утверждаются по ходу)

### Десктопное приложение
- **Shell**: Tauri (Rust backend + WebView)
- **UI**: React + TypeScript
- **Node-редактор**: собственный Canvas-движок
- **Viewport**: WebGPU (превью) / Vulkan (продакшн рендер)

### Бэкенд / Облако
- **API**: Rust (Axum)
- **Auth**: Keycloak
- **Очереди рендера**: Redis + BullMQ
- **Хранилище**: S3-compatible + PostgreSQL
- **Real-time**: CRDT (Yjs) через WebSockets

### Аудио (ШИНА)
- **DSP-движок**: JUCE (C++)
- **Пространственный звук**: собственная реализация
- **Протоколы**: кириллические расширения (утверждаются по ходу)
